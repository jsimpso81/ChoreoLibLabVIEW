Choreo_Trajectory_Sample

Return an interpolated sample of the trajectory at the given timestamp.



Inputs: 

  -  ChoreoTrajectory  -  cluster  -  The trajectory to sample.

  -  timestamp  -  double  -  The timestamp of this sample relative to the beginning of the trajectory.

  -  mirrorForRedAlliance  -  boolean  -  whether or not to return the sample as mirrored across the field midline (as in 2023).  (Optional.  Default: false )



Outputs:

  -  ChoreoTrajectoryState  - cluster - The ChoreoTrajectoryState at the given time.
