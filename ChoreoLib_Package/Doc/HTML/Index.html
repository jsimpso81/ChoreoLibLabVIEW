ChoreoLib LabVIEW Library

Introduction

The ChoreoLib LabVIEW library provides utility functions to read, sample, and follow trajectories created by Choreo. The library can read trajectory files created by the GUI. It can then follow these trajectories.

The library has an extensive set of examples that can be found under the LabVIEW "Find examples..." menu.

The library source code, package build specifications, and test package can be found here https://github.com/jsimpso81/ChoreoLabVIEW
